Easily send single or bulk SMS via Bible - Africa's Talking SMS. Manage recipients and customize messages with secure integration from WordPress.

== Description ==

The Bible - Africa's Talking SMS Plugin allows your WordPress site to send both single and bulk SMS messages using the Africa's Talking API. This plugin integrates seamlessly with Africa’s Talking SMS gateway, providing a reliable and secure method for communicating with your audience directly via SMS. Whether for announcements, notifications, marketing, or any other purpose, you can easily manage and send messages to individuals or groups through your WordPress admin panel.

== Information about Africa's Talking API ==
The following APIs are involved in this plugin: https://africastalking.com/

== Features of the plugin include:== 

* Send single or bulk SMS to multiple recipients.
* Secure and efficient integration with Africa’s Talking API.
* Manage message history and recipient lists through the WordPress dashboard.
* Compatible with other WordPress plugins and themes.
* Supports localization and exception handling for improved usability.
* Delivery Reports: Track the status and delivery reports of sent messages.

== Frequently Asked Questions ==

= How do I install the Send SMS Plugin? =

Install the plugin via the WordPress Admin Dashboard by navigating to Plugins > Add New, then upload and activate the plugin

= How do I access the plugin settings? =

Go to the plugin settings page under the Settings or SMS menu in the WordPress Admin Dashboard. Enter your SMS gateway credentials (such as API key and secret) and configure other settings as needed.

= Can I send SMS to multiple recipients at once? =

Yes, the plugin supports bulk SMS sending. You can upload a CSV file containing multiple phone numbers for sending bulk SMS.
