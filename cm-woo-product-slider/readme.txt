== Description ==

**C-Metric Product Slider & Carousel for WooCommerce** lets you display your WooCommerce products beautifully in a responsive and customizable carousel or slider. Perfect for showcasing your featured, best-selling, or specific-category products on any page or post.

**Key Features:**
- Display WooCommerce products in an eye-catching carousel or slider.
- Fully responsive and mobile-friendly layout.
- Options to display featured, best-selling, latest, or custom category products.
- Multiple customization options for the number of slides, autoplay, pagination, and navigation controls.
- Easy to use with a simple shortcode.

== Frequently Asked Questions ==

= How do I install the C-Metric Product Slider & Carousel for WooCommerce plugin? =

You can install the C-Metric Product Slider & Carousel for WooCommerce plugin via the WordPress admin panel by navigating to Plugins > Add New, searching for C-Metric Product Slider & Carousel for WooCommerce, and clicking Install Now. After installation, click Activate to enable the plugin
